How to Use:

  1. Upload an image using the file button
  2. Click on the image to pick colors with the eyedropper
  3. Click on a palette cell once to set the FIRST color
  4. Pick another color and click the same cell to set the SECOND color
  5. Once both colors are set, click and drag within the cell to blend them smoothly
  6. The blending transitions from left color to right color as you drag across
  7. Save or export your color palette when done
