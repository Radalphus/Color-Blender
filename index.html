How to Use:

  1. Upload an image using the file button
  2. Click on the image to pick colors with the eyedropper
  3. In Color Picking Mode: Click a cell once to set the FIRST color, then pick another color and click again to set the SECOND color
  4. Switch to Blending Mode using the mode toggle buttons
  5. In Blending Mode: Drag your mouse across the cell to manually blend the two colors together
  6. The colors mix together like real paint - keep dragging to blend more!
  7. The final result will be a single mixed color (not a gradient)
  8. All blending is permanent - your work is saved to the canvas!
  9. Save or export your color palette when done
